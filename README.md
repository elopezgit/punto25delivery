# punto25delivery

Catálogo web de pedidos auto-gestionados por WhatsApp para **Punto 25 Delivery** (Granja, Rebozados y Congelados), ubicado en Corrientes 664, Barrio Norte, San Miguel de Tucumán.

Este proyecto es una plataforma interactiva, elegante y rápida adaptada a dispositivos móviles que permite a los clientes armar sus pedidos, personalizar las cantidades de productos pesables (Kilo o Medio Kilo) y enviarlos formateados de manera automática a los celulares de contacto oficiales del negocio.

---

## 🚀 Características Clave

- **🎨 Identidad de Marca de Elite:** Colores corporativos (Celeste brillante, Azul marino, Amarillo) con tipografía minimalista de alta gama (*Outfit*) que transmiten simpleza y sofisticación.
- **⚖️ Soporte Avanzado de Ventas por Peso:** Control bidireccional que permite a los usuarios seleccionar entre `½ Kg` y `1 Kg` de forma directa en las tarjetas de productos, permitiendo que convivan de manera independiente en el carrito de compras.
- **📲 Carrusel de Promociones con Scroll Inteligente:** Los dots indicadores del banner de promociones son clickeables, deslizando automáticamente el carrusel en cualquier dispositivo o computadora.
- **🆔 Ticket de Pedido Único (#25-XXXX):** Generación alfanumérica aleatoria de un ID de orden para optimizar la contabilidad, archivo y seguimiento para el cliente y el negocio.
- **👥 Selector de Canal de WhatsApp:** El carrito permite al usuario enviar el pedido a cualquiera de los dos números provistos en el catálogo original, equilibrando la recepción del trabajo.
- **⚡ Micro-animaciones y Transiciones:** Efecto de partículas dinámicas de los productos volando al carrito y carga splash mockup del logo inicial con CSS nativo.
